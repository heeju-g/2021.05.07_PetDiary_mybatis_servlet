### 프로젝트 전체 기능     

<details>
<summary><b>전체 기능 설명 펼치기</b></summary>
<div markdown="1"> </br>  
   - 로그인/회원가입 – 로그인(소셜로그인 지원) , 회원가입 및 아이디 비밀번호 찾기, 이메일 인증 기능 </br>    
   - 캘린더 - 해당 날짜의 일정 확인, 추가, 수정, 삭제하는 기능   </br>  
   - 결제 - 예약 수수료 결제 기능, 화상채팅 진료시간에 따른 결제, 업체 서비스 이용료 결제   </br>  
   - 게시판 - 병원/식당/여행/정보공유 게시판 글 작성 수정, 삭제, 댓글작성, 선택, 확인 등의 기능   </br>  
   - 예약 - 예약 페이지 리스트 구현, 문자발송, 예약기능   </br>  
   - 닮은 동물 찾기 - 머신러닝(특정 동물과 닮은 연예인 사진 이용)을 통해 실제 사용자의 웹캠 혹은 사진 업로드 시, 닮은 동물을 찾아주는 기능    </br>  
   - 날씨 - 원하는 지역 날씨정보 제공   </br>  
   - 지도 - 가게 및 병원 등의 위치 표기 및 거리, 이동시간 확인기능. 데이터베이스와 연결하여 저장된 업체 위치 표기 및 검색기능   </br>  
   - 비속어 필터링 - 게시글에서 비속어 필터링 기능을 통해 해당 게시물을 읽어들여 욕설로 판단되는 단어를 특수문자로 변경   </br>  
   - 화상채팅 - 원격 진료 기능 구현   </br>  
   - 챗봇 - AI 대화처리를 통해 병원 예약과 반려동물 관련 질의응답 기능   
</div>
